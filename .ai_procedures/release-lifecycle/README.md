Follow these procedures when:
- Planning a new release
- Preparing release candidates
- Deploying to production
- Managing post-release activities

## Related Lifecycles

- **Feature Lifecycle** - Releases include completed features
- **PR Lifecycle** - PRs are merged into releases
- **Incident Lifecycle** - May be triggered by release issues
- **Change Management Lifecycle** - Releases require change approval
